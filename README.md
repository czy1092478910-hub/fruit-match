# 水果连连看

休闲益智类连连看游戏，适老设计：节奏轻快、界面清晰、不刺激。目标用户为老年人。

## 玩法

- 点击两个**相同**水果，若能用不超过**两折**的线连起来（可经棋盘外绕行），即可消除。
- 每关有倒计时，消除一对会加时；2 秒内连续消除有连击加时。
- 支持**提示**、**洗牌**、**重新开始**。

---

## 音乐说明

- **背景音乐（BGM）**：每关自动播放轻快、不刺激的背景音乐。
  - 共 5 个曲目池，按关卡轮换（第 1 关对应曲目 1，第 2 关对应曲目 2……第 6 关再回到曲目 1）。
  - **不依赖外部文件优先**：若未提供 `assets/bgm1.mp3`～`assets/bgm5.mp3`，会使用 **Web Audio API 合成**的简单旋律循环，保证无网络也能播放。
  - 若放置了 `assets/bgm1.mp3` 等文件，会优先加载对应关卡 MP3；**加载失败时自动回退到合成 BGM**。
- **音乐开关**：界面左侧有「🔊 音乐开 / 🔇 音乐关」按钮，可随时开关 BGM。
  - 默认开启；**关闭后刷新页面或进入下一关都会记住“关”**（使用 localStorage）。
- BGM 音量比点击、消除等音效更小（约 0.2），避免盖过操作反馈。

---

## 暂停说明

- 左侧有 **「⏸ 暂停」** 按钮，点击后：
  - 倒计时**停止**（不扣时间）；
  - 棋盘**不可点击**，无法操作；
  - 棋盘区域会出现**半透明遮罩**与「已暂停」文字，**避免偷看记位置**。
- 再次点击按钮（此时显示「▶ 继续」）或按键盘 **Esc** 可恢复游戏。
- 提示、洗牌在暂停状态下不可用；重新开始、音乐开关仍可用。

---

## 路径折线说明

- 消除时显示的**连接线**为**真实路径**：与规则一致，可能是直线、一折或两折的折线。
  - 连接算法会返回路径上的关键点（起点、拐点、终点），例如 `[p1, corner1?, corner2?, p2]`。
- 使用 **SVG** 在棋盘上方绘制折线，线条经过**格子中心点**，与 UI 缩放一致。
- 线条带**发光与渐隐**效果，显示约 250～400ms 后自动消失，不遮挡后续操作。

---

## 技术说明

- 纯原生实现（无大型框架），兼容桌面与手机。
- 按钮最小点击区域 ≥ 48px，便于老年人触控。
- 动画控制适量，减少重排，保证流畅。

---

## 调试开关说明（Ctrl+D）

- 按 **Ctrl + D** 可切换显示/隐藏**调试面板**（默认隐藏，不影响正常玩家）。
- 或连续点击标题「水果连连看」**5 次**（2 秒内）也可打开调试面板。
- 面板中会实时显示：当前缩放、视口尺寸、棋盘容器信息、是否溢出/裁切、连线层尺寸、最近一次连线信息等；并提供「复制诊断信息」「重置缩放」「强制重绘连线层」按钮，便于自检与反馈问题。

---

## 通关页说明

- 通过**第 10 关**后，会弹出**全屏通关页**（覆盖整个游戏界面），并播放彩带/烟花动画与**掌声音效**。
- 掌声音效优先使用 **Web Audio 合成**（无需外部文件）；若需使用音频文件，可放置 `assets/clap.mp3`，并在代码中切换为播放该文件（当前版本仅使用合成音）。
- 通关页左下角有**制作者照片轮播**：请将 **jpg 照片**放在与 `index.html` 同目录下，命名为 `me1.jpg` 与 `me2.jpg`，每 2.5 秒淡入淡出切换。
- 点击**【再玩一遍】**会关闭通关页并重置到第 1 关。
- 若浏览器因自动播放策略未播放掌声，页面上会出现「点击任意处播放声音」提示，点击页面空白处即可播放。

### 开发者快速预览（不用打到最后一关）

- 按 **Shift + P** 可直接打开通关页预览（不改变当前关卡与进度）。

---

## 运行

用浏览器直接打开 **`index.html`** 即可看到效果；若需本地服务器（例如用于加载图片/音频），可：

```bash
npx serve .
```

然后将浏览器访问提示的地址。
